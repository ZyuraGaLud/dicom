医用画像処理Viewer 操作マニュアル
1. 概要
本ソフトウェアは、CTまたはMRIの画像系列データ（DICOM形式）を読み込み、3断面（Axial, Sagittal, Coronal）の同時表示および階調調整を行うためのビューアです 。 大阪公立大学工業高等専門学校 知能情報実験実習2「医用画像処理実験」の最終課題として作成されました 。

2. 動作環境・実行方法

特殊な環境（Pythonやライブラリ）のインストールは不要です 。

Windows環境であれば、viewer.exe をダブルクリックするだけで起動します。

3. 使用方法

データの読み込み: 画面右上の「DICOMフォルダを選択」ボタンを押し、CTデータが含まれるフォルダを選択します 。

画像の調整:


スライス位置: 「Axial/Sagittal/Coronal」それぞれのスライダーを動かすことで、断層位置を自由に変更できます 。


階調調整: 「Window Width (WW)」および「Window Level (WL)」スライダーで、観察目的に適したコントラストと明るさに調整可能です 。


情報の確認: 画面右側のパネルに、DICOMヘッダから取得した以下の情報が表示されます 。

画像の縦・横サイズ 

スライス厚 

スライス数 

4. 主な機能（課題要件の充足）

3断面同時表示: Axial（原画像）をメインとし、Sagittal、Coronalを同時に表示・操作できます 。


スライス面表示: 各画像上に、他断面の切断位置を示すガイドライン（スライス面）を表示しています 。


スライス番号表示: 各画像左上に、現在のスライス番号をリアルタイムで表示します 。
