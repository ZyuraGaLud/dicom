# 医用画像処理Viewer 操作マニュアル

## 1. 概要

本プロジェクトは、大阪公立大学工業高等専門学校 知能情報実験実習2「医用画像処理実験」の最終課題として作成されたビューアです 。
CTまたはMRIの画像系列データ（DICOM形式）を読み込み、3断面（Axial, Sagittal, Coronal）の同時表示および階調調整が可能です 。

## 2. 動作環境・配布形式

「特殊な環境（ソフトやライブラリ）が必要とならないように」という条件  に基づき、実行ファイル形式（.exe）を作成しました。

* **実行ファイル形式**: GitHubの容量制限（100MB）により、exe本体は外部ストレージ（[ここにGoogleドライブ等のリンクを貼る]）に配置しています。このexeを使用すれば、Pythonのインストールなしで動作します。
* **ソースコード実行**: Python 3.11環境で `pip install pydicom numpy Pillow` を実行することで動作可能です。

3. 提出物 

* 
`viewer.py`: プログラムソースコード 


* 
`viewer.exe`: 実行ファイル（上記URLより参照） 


* 
`README.md`: 本操作マニュアル（Markdown形式） 



## 4. 主な機能（課題要件の充足）

スライドに記載されたすべての条件を満たしています：

* **画像表示と操作**:
* 
**3断面表示**: Axial（原画像）を必ず表示しつつ、Sagittal、Coronalを同時に表示・切り替え可能です 。


* 
**スライダー操作**: Window Width (WW), Window Level (WL), スライス画像をスライダで直感的に変更できます 。


* 
**スライス番号**: 画像上に現在のスライス番号を表示します 。


* 
**スライス面表示**: 各画像上に、他断面との交差位置（スライス面）を表示します 。




* 
DICOMヘッダ情報の取得表示 :


* 画像の縦・横サイズ 


* スライス厚 


* スライスの数 





## 5. 使用方法

1. 
**データの読み込み**: 画面の「DICOMフォルダを選択」ボタンを押し、CTデータ（.dcmファイル群）が含まれるフォルダを選択します 。


2. 
**画像の調整**: 右側のパネルにある各スライダーを動かし、断層位置や階調を調整します 。
